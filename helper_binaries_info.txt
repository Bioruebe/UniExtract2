Legend
---------
Special features for several extractors:
Success evaluation		means that the console output is checked for success messages, not the output directory size/time, 
						so there will be no failure message when overwriting files
Fast detection			means that the filetype is recognized be the unix file tool, which is slightly faster than TrID
Percentage indicator 	means that the progress is shown in the status box (in percent)	otherwise filesize of output directory is shown

--------- 				means same as above
x						means yes, without exeptions
o						means partly
-						means no
!						means planned in future releases
*						means used with permission

Thanks to all developers for their great tools!
